# MATPLOTLIB

## ЁЯУИ Matplotlib тАУ Basics

- `matplotlib.pyplot` тЖТ plotting module for visualization  
- `plt.plot()` тЖТ creates line plots  
- `color` тЖТ defines plot color  
- `linewidth` тЖТ controls line thickness  
- `linestyle` тЖТ defines line pattern/style  
- `xlabel()` тЖТ labels x-axis  
- `ylabel()` тЖТ labels y-axis  
- `title()` тЖТ adds plot title  

---

## ЁЯУИ Matplotlib тАУ Formatting & Styling

- Format Strings тЖТ shorthand styling (e.g., `"g--*"`)  
- Marker (`marker`) тЖТ defines point style  
- Linestyle (`linestyle`) тЖТ controls line pattern  
- Color (`color`) тЖТ supports named and hexadecimal colors  
- `markersize` тЖТ controls marker size  
- `alpha` тЖТ adjusts transparency level  
- Formatting can be passed directly as arguments in `plt.plot()`
