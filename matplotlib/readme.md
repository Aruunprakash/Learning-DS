# MATPLOTLIB

## ЁЯУИ Matplotlib тАУ Basics

- `matplotlib.pyplot` тЖТ plotting module for visualization  
- `plt.plot()` тЖТ creates line plots  
- `color` тЖТ defines plot color  
- `linewidth` тЖТ controls line thickness  
- `linestyle` тЖТ defines line pattern/style  
- `xlabel()` тЖТ labels x-axis  
- `ylabel()` тЖТ labels y-axis  
- `title()` тЖТ adds plot title  

---

## ЁЯУИ Matplotlib тАУ Formatting & Styling

- Format Strings тЖТ shorthand styling (e.g., `"g--*"`)  
- Marker (`marker`) тЖТ defines point style  
- Linestyle (`linestyle`) тЖТ controls line pattern  
- Color (`color`) тЖТ supports named and hexadecimal colors  
- `markersize` тЖТ controls marker size  
- `alpha` тЖТ adjusts transparency level  
- Formatting can be passed directly as arguments in `plt.plot()`  

---

## ЁЯУИ Matplotlib тАУ Multiple Lines & Legends

- Plot multiple lines in a single graph  
- Label lines for identification  
- `legend()` тЖТ displays labels on plot  
- `loc` тЖТ sets legend position (`best`, `upper right`, etc.)  
- `shadow` тЖТ adds shadow to legend  
- `fontsize` тЖТ controls legend text size  
- `grid()` тЖТ adds background grid for better readability  

---

## ЁЯУК Matplotlib тАУ Bar Graphs

- `plt.bar()` тЖТ creates vertical bar graph  
- `plt.barh()` тЖТ creates horizontal bar graph  
- `xticks()` тЖТ customizes x-axis labels and positions  
- `np.arange()` тЖТ generates index positions for bars  
- `width` тЖТ controls bar thickness/width  

---

## ЁЯУК Matplotlib тАУ Histogram

- `plt.hist()` тЖТ creates histogram for data distribution  
- `bins` тЖТ defines number/range of intervals  
- `rwidth` тЖТ controls relative bar width  
- `color` тЖТ sets histogram color  
- `histtype` тЖТ defines histogram style  
- `orientation` тЖТ sets vertical or horizontal layout
