# MACHINE LEARNING

## 📌 Overview
Machine Learning is a branch of Artificial Intelligence that enables systems to learn patterns from data and make predictions or decisions. It uses mathematical models and algorithms as the foundation, while Deep Learning is a subset of Machine Learning that uses neural networks to handle complex data and tasks.

---

### 🧰 Scikit-Learn (`sklearn`)
- Popular Python library for Machine Learning  
- Provides tools for data preprocessing, models, and evaluation  

---

### 📐 Linear Model
- Mathematical model representing relationship between variables  
- Linear Regression follows the form: dependent variable = slope × input + intercept  

---

## 📈 Simple Linear Regression

- Simple Linear Regression → predicts a dependent variable using one independent variable  
- Used to find relationships between variables through a straight-line model
